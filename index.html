🆘 SOS Cervicalgies & Dos

Spécial longue route

🦴 Cervicales (Post-Conduite)

  • Le "Menton-Poitrine" : Dos droit, amène le menton vers le sternum. Tiens 30s. Libère la nuque.
  • Inclinaison latérale : Oreille vers l'épaule (sans lever l'épaule). 30s de chaque côté.

🧘 Étirements Tapis (Full Body)

  • Chien tête en bas : Pour étirer toute la chaîne postérieure (dos + jambes) après le trek.
  • L'Enfant (Child Pose) : Idéal dans le van pour relaxer les lombaires avant de dormir.
  • Posture du Pigeon : Ouvre les hanches fermées par la position assise prolongée en van.

🚐 Coaching en Van

  • 📍 Spot Check : Utilise les tables de pique-nique pour tes pompes ou tes fentes bulgares.
  • 🎒 Matos : Ton sac photo de 12kg est ton meilleur haltère. Goblet Squats et Rowing avec le sac.
  • ❄️ Récup : Les lacs norvégiens sont parfaits pour une séance de cryothérapie (bain froid) naturelle !
